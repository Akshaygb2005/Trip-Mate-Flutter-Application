# Fare Calculator 

A Flutter-based mobile app that helps users calculate fares easily.  
Built with **Firebase**, Flutterâ€™s native **Navigator** for screen routing. , and **Cloud Firestore** for data storage.  

---

##  Features
- Splash screen with smooth navigation
- Firebase integration
- Cloud Firestore database
- state management
- Simple and clean UI

---
##  Tech Stack
- **Frontend**: Flutter (Dart)
- **State Management**: Stateful Widgets (`setState`)
- **Navigation**: Flutter Navigator
- **Backend**: Firebase Authentication & Firestore
- **Database**: Cloud Firestore
